Remote Desktop (RDP) Full Connection Guide

This guide explains in detail how to connect to your Windows VPS / Server using Remote Desktop (RDP) from Windows, Mobile Phone, and macOS.

1. Remote Desktop Using Windows (PC / Laptop)

Required Application:
Remote Desktop Connection (Built-in in Windows)

Step-by-Step Instructions

  1. Press Windows + R on your keyboard
  2. Type mstsc and press Enter
  3. In the Computer field, enter your Server IP or Domain
  4. Click Connect
  5. When prompted, enter your:
    • Username
    • Password
  6. Click OK to proceed
If a security warning appears, click Yes to continue.

You are now successfully connected to your Windows VPS.

2. Remote Desktop Using Mobile Phone (Android / iPhone)

Required Application (Mobile):
Windows App by Microsoft Corporation
  • Android: Google Play Store
  • iPhone / iPad: Apple App Store

Previously known as “Microsoft Remote Desktop”.

Step-by-Step Instructions

  1. Download and install Windows App on your phone
  2. Open the app
  3. Tap the + icon
  4. Select Add PC
  5. Enter your Server IP or Domain
  6. Add your Username and Password
  7. Save the configuration
  8. Tap the saved server profile to connect

The Windows desktop will now appear on your mobile screen.

3. Remote Desktop Using macOS (MacBook / iMac)

Required Application:
Microsoft Remote Desktop (macOS)

Step-by-Step Instructions

  1. Open the Mac App Store
  2. Search for Microsoft Remote Desktop
  3. Install and open the application
  4. Click Add PC
  5. Enter your Server IP or Domain
  6. Enter your Username and Password
  7. Click Connect

Your Windows VPS desktop will now be displayed on your Mac.

Important Notes

  • Ensure your service status is Active
  • Always use the correct Username & Password
  • A stable internet connection is required
  • If login fails repeatedly, please contact support
